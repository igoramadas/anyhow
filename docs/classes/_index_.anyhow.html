Properties

compact

compact: boolean = true
  • Defined in index.ts:42

Messages will be compacted (spaces and breaks removed), default is true. Set to false to log original values including spaces.

errorStack

errorStack: boolean = false
  • Defined in index.ts:48

Log error stack traces? Default is false. Use it with care! Set to true to add stack traces to the log output.

levels

levels: string[] = ["info", "warn", "error"]
  • Defined in index.ts:54

Array that controls which log calls are enabled. By default it's info, warn and error, so debug won't log anything.

preprocessor

preprocessor: Function
  • Defined in index.ts:83

A function(arrayOfObjects) that should be called to preprocess the arguments. This is useful if you wish to remove or obfuscate data before generating the logging message. Can either mutate the passed arguments or return the processed array as a result.

separator

separator: string = " | "
  • Defined in index.ts:60

Set the separator between arguments when generating logging messages. Default separator is a pipe symbol " | ".

setup

  • setup(lib?: string | any, options?: any): void
    • Defined in index.ts:191

    Setup will try to load compatible loggers, and fall back to the console if nothing was found. You can also force a specific library to be loaded passing the object directly or as string "winston|bunyan|console". Set "none" to disable logging.

    Parameters

    • Optional lib: string | any

      Optional, force a specific library to be used. If not passed, will try winston then bunyan then console.

    • Optional options: any

      Additional options to be passed to the underlying logging library.

    Returns void
